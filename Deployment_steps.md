# Faculty Document Management System

## Requirements
- PHP 8+
- Composer
- MySQL
- Node.js

## Backend Setup
1. cd backend
2. composer install
3. copy .env.example to .env
4. configure DB
5. php artisan key:generate
6. php artisan storage:link
7. php artisan serve

## Database
Import faculty_dms.sql into MySQL

## Frontend Setup
1. cd frontend
2. npm install
3. npm run build
